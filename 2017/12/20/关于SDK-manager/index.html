关于SDK manager

Posted on 2017-12-20

前言

最近感觉诸事不顺，昨天eclipse的SDK manager一直打不开，很是奇怪，在我一番探索之下，终于解决，下一步就是解决google play services的问题了。为了记录，写下这篇博客。

现象

突然之间，打不开eclipse的SDK manager了，给出的报错如下，

网上得票最高的解答居然是缺少traceview.bat文件，然后给出了下载网址，下载完放到文件夹中之后并没有解决问题，一直在怀疑自己。

原因

更深入的研究之后，发现原因是这样，eclipse和Android studio用的是同一个SDK，而Android studio导致之前的SDK文件夹里面缺失了很多文件，eclipse用不了这样的SDK。

解决方法

重新下载SDK，将新的SDK与eclipse绑定。
